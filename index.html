Tables

Tables let you arrange data into rows and columns.
Tables consists of table cells inside rows and columns.
Table cells can contain all sorts of HTML elements.
Such as: text, images, links, lists, more tables, etc.

HTML Explanation
<td> Each cell uses a <td> and a </td> tag.
td can consist of basically any HTML element.
td = table data.
<tr> All td in a table must be surrounded by a
<tr> and a </tr> tag in order to
create a new row in the table.
tr = table row.
<th> A table's first row is composed of
a <th> and a </th> tag.
Creates a header for a column in the table.
th = table head.

CSS used

Border creation is all up to the CSS code.
Using an external stylesheet it looks like this:
table, th, td {
border: 1px solid black;
border-collapse: collapse;
text-align: center;
font-size: larger;
}

Work

border-collapse turns the double lined table into a single lined one.
text-align aligns the text to the parent element.
font-size changes the size of the font.
Without all of this, it ends up looking like elements spaced out on a doc.
